# Portfolio Website Clone

Modern portfolio website built with Next.js, React, and Tailwind CSS.

## 🚀 Features

- ✨ Modern glassmorphism design
- 🎨 Smooth animations and transitions
- 📱 Fully responsive design
- 🌙 Dark mode support
- ⚡ Optimized for performance
- 🎯 HR-friendly professional layout

## 🛠️ Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI
- **Icons:** Lucide React
- **Deployment:** GitHub Pages

## 📦 Installation

1. Clone the repository:
\`\`\`bash
git clone https://github.com/your-username/portfolio-website-clone.git
cd portfolio-website-clone
\`\`\`

2. Install dependencies:
\`\`\`bash
npm install
\`\`\`

3. Run the development server:
\`\`\`bash
npm run dev
\`\`\`

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🚀 Deployment

### GitHub Pages

1. Update the `basePath` and `assetPrefix` in `next.config.mjs` with your repository name.

2. Build and deploy:
\`\`\`bash
npm run build
\`\`\`

3. The site will be automatically deployed to GitHub Pages via GitHub Actions.

### Manual Deployment

\`\`\`bash
# Build the project
npm run build

# The static files will be in the 'out' directory
# Upload the contents of 'out' to your hosting provider
\`\`\`

2. **Styling:** Modify `tailwind.config.js` and component styles
